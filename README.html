Уроки по Selenium. Python

Полезные ссылки:

  • Chromedriver
  • Geckodriver - для браузера Mozilla Firefox

  • Список опций Chromium
  • Библиотека fake-useragent

Для запуска сделайте следующее (Для Ubuntu/Debian):
  • Скопируйте проект:
    https://github.com/dev2033/python_selenium_tutoral.git

  • Создайте виртуальное окружение и активируйте его:
    python3 -m venv venv
    source venv/bin/activate

  • Установите зависимости:
    pip install -r requirements.txt
Также нужно создать переменные окружения со своим логином и паролем в ВК
